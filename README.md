# HotelRoomProgram
שאלה 2: 
בבית מלון מסוים יש 20 קומות. בכל קומה יש לכל היותר 100 חדרים. מספר החדר נקבע כך:
שתי הספרות הימניות מייצגות את המרחק מהמעלית. זאת אומרת, ככל ששתי הספריות הימניות נמוכות יותר, כך החדר קרוב יותר למעלית. 
הספרות הבאות מייצגת את מספר הקומה. 
לדוגמה: 
חדר מספר 1223 נמצא בקומה מספר 12, ומרוחק 23 חדרים מהמעלית. 
חדר מספר 801 נמצא בקומה מספר 8 ומרוחק חדר אחד מהמעלית. 
חדר מספר 1500 נמצא בקומה מספר 15 והוא הקרוב ביותר למעלית בקומה זו. 
סעיף א': 
כתבו את המחלקה Room המייצגת חדר במלון. 
במחלקה יהיה שדה מסוג מספר שלם בשם number, המייצג את מספר החדר. 
כמו כן יהיה שדה מסוג מספר שלם בשם type, המייצג את סוג החדר (1 – רגיל, 2 – משודרג, 3 – סוויטה). 
בנוסף, יהיה שדה בוליאני בשם available, המייצג האם החדר תפוס נכון לעכשיו. 

סעיף ב': 
הוסיפו למחלקה את המתודה הבאה: 
int floor()
המתודה מחזירה את מספר הקומה. 


סעיף ג': 
מחיר החדר נקבע באופן הבא: 
המחיר ההתחלתי הוא 2000 שקלים ללילה. 
בעשר הקומות הראשונות אין תוספת תשלום, אך החל מהקומה ה-11, כל קומה מוסיפה לעלות של המחיר עוד 100 שקלים. כלומר, העלות הנוספת של חדר בקומה ה-14, היא 400 שקלים. 
חמשת החדרים הקרובים ביותר למעלית מוסיפים 200 שקלים לעלות. 
אחרי שמחשבים את כל העלות הזאת, יש להוסיף לעלות את סוג החדר: 
אם מדובר בחדר משודרג, העלות גדלה פי 1.5. אם מדובר בסוויטה, העלות גדלה פי 2.5. 
לדוגמה: 
חדר משודרג בקומה ה-13, שנמצא קרוב למעלית יעלה: 
2000 (מחיר בסיסי) 
ועוד 3*100 (עבור קומה 13)
ועוד 200 שקלים (עבור קרבה למעלית)
2000+300+200 = 2500. 
אבל כיוון שהחדר משודרג, נכפיל את העלות כפול 1.5 ונקבל 3750. 

הוסיפו למחלקה את המתודה int calculatePrice() המחזירה את מחיר החדר. 

סעיף ד': 
כתבו מחלקה בשם Hotel ובה השדות:
-String name – מייצג את שם המלון
Room[] rooms המייצג את רשימת החדרים במלון. הוסיפו למחלקה מתודה המקבלת מספר שלם type ומספר שלם budget. 
המתודה תחזיר רשימה של חדרים שעומדים בתנאים הבאים:
-הם פנויים
-הם מסוג type
-העלות שלהם לא עולה על price. 

סעיף ה': 
הוסיפו למחלקה Hotel מתודה בשם int income() המחשבת את ההכנסה הנוכחית של המלון. ההכנסה הנוכחית היא העלות הכוללת של כל החדרים במלון שתפוסים (אינם פנויים) כרגע. 

סעיף ו': 
כתבו מחלקה ראשית והוסיפו בה מתודה בעלת החתימה הבאה: 
String mostProfitable(Hotel[] hotels)
המתודה מחזירה שמו שם המלון הרווחי ביותר (זה שההכנסות הנוכחיות שלו הן הגבוהות ביותר).
